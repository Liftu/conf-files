# Personal conf files

## Bashrc

```bash
cp bashrc ~/.bashrc
```

## Powerline

```bash
pip install --user git+git://github.com/powerline/powerline
```

## ipython

```bash
cp ipython_config.py ~/.ipython/profile_default/ipython_config.py
```

## tmux

```bash
cp tmux.conf ~/.tmux.conf
```
